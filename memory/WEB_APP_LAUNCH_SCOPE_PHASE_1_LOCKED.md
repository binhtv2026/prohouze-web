# WEB / APP LAUNCH SCOPE - PHASE 1 LOCKED

## Muc tieu

Khoa scope dot 1 cho ProHouzing de toan bo quyet dinh tiep theo bam vao mot nguon su that duy nhat:

- module nao mo ngay
- module nao chi noi bo
- module nao de pha sau

Khong con trien khai theo kieu mo toan bo roi moi don dep sau.

## Ket luan chot

### Mo dot 1

- Sales app
- CTV / Dai ly app
- Manager hybrid workspace
- BOD hybrid workspace
- Marketing hybrid workspace
- Admin / System web
- Finance web
- HR web
- Legal web
- Website / CMS web

### Chi noi bo

- Inventory / Pricing noi bo
- Automation / Email noi bo

### Pha sau

- Advanced BI / phan tich sau
- Customer portal / app khach hang

## Nguyen tac

1. Dot 1 chi mo nhung cum tao gia tri van hanh truc tiep.
2. Cum nao chua ro surface hoac chua ro owner thi khong mo.
3. App chi mo cho luc luong tien tuyen va role can thao tac nhanh.
4. Web giu toan bo quan tri, back office va module bang lon.
5. Hybrid chi danh cho role thuc su can ca hai mat tran.
